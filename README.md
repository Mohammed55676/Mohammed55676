## 🚀 Featured Projects

### 🔹 Menassat Al-Khair — منصة الخير
A full-stack web platform that bridges the gap between donors and those in need through financial campaigns, in-kind donations, and real-time communication. The platform enforces a charity verification workflow, privacy-first messaging via Socket.IO, and an admin dashboard to oversee all activity.

**Tech Stack:** React 19, Vite, Tailwind CSS v4, Node.js, Express.js, MongoDB, Socket.IO, JWT, Firebase Auth  
**Live Demo:** [al-khairplatform.netlify.app](https://al-khairplatform.netlify.app/) | **Repository:** [View Code](https://github.com/Mohammed55676/Donation-Platform)

**Key Features:**
- 🔐 Secure auth with OTP email verification & Google OAuth
- ✅ Charity verification workflow with admin approval
- 💬 Real-time Socket.IO chat with role-based privacy controls
- 💰 Monetary campaigns with progress tracking
- 📦 In-kind (item) donations with condition tracking
- 🙋 Volunteer opportunities & community requests
- 🛡️ Admin dashboard for user/charity/report management

### 🔹 Skyline Glass — Marketing Website
Marketing website for Skyline Glass, a Durham, NC glass installation contractor. A fully responsive multi-page static site featuring services, a media gallery, contact form, and quote request — built solo with pure HTML, CSS, and no frameworks.

**Tech Stack:** HTML5, CSS3, CSS Grid/Flexbox, Font Awesome, Google Fonts, Netlify  
**Live Demo:** [skylineglass.netlify.app](https://skylineglass.netlify.app/) | **Repository:** [View Code](https://github.com/Mohammed55676/Blue-Ray-Glass)

- 🏠 Home, About, Services, Media Gallery, Contact & Get Quote pages
